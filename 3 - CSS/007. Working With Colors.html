The Basic Langauge of the Web: HTML

Posted by Laura Jones on Monday, Jue 21st 2027

All Modern website and web applications are built using three fundamental [emphsize] technologies: HTML, CSS and JavaScript. These are the language of the web.

What is HTML?

HTML stands for HyperText Markup Language. It's a markup langauge that web developers use to structure and describe the content of a webpage (not a programming language).

HTML consiste of elements that describe different types of content: paragraphs, links, headings, images, video, etc. Web browsers understand HTML and render HTML as websites.

in HTML. each element is made up of 3 parts

  1. The opening tag
  2. The closing tag
  3. The Actual Element

You can learn more at MDN Web Docs
