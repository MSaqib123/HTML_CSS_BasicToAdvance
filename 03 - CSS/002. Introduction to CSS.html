CSS

Cascading Style sheets

CSS describes the Visual style and presentation of the Content written in HTML


Parts of CSS

  • Selector ===> h1,h2,h4,div,.className, #idName
  • Decorative block ====> #className{blockof code}
  • Property Name ====> background-color , font-size , color ,....
  • Property Value ====> red , 25px , ......
  • Declaration style color:red , background-color:green, font-size:24px
  • Example:
  • .ClassName{
      font-color:red,
      font-size:14px
    }
